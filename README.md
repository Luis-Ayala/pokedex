# Getting Started

### Install required jars 

```console
mvn clean install
```

### Run pokeapp springboot based project

```console
mvn spring-boot:run -pl pokeapp
```

### Testing with SoapUI

- Open SoapUI and then import the xml file [SoapUI file](./docs/pokedex-soapui-project.xml).
- Open a request and test it.
